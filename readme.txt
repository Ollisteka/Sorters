Сортировки.
---
Версия 1.3
Автор: Жукова Ольга (zhukova.o.m@yandex.ru)
---
В данной программе присутствуют 5 алгоритмов сортировки, с поддержкой выбора ключа для сравнения и
вывода в обратном порядке. Есть возможность узнать количество совершённых сравнений и перестановок.
---
Консольная версия

Справка по запуску:
    Windows: python sort.py --help
    Linux: python3 sort.py --help
Пример запуска: python sort.py --shell test.txt

---
Генератор массивов
---
Позволяет сгенерировать массивы для дальнешего использования. Массивы хранятся в словарях (лучший,
худший и рандомный вариант)
---
Генератор данных для построения графиков
---
Используя генератор массивов, программа прогоняет все данные через все функции сортировок, заполняя
словарь числом сравнений и перестановок, а также временем выполнения сортировки. Этот словарь сохраняется
в файлы с расширением .csv, который затем можно обработать, используя программу OriginPro

Справка по запуску:
    Windows: python data_generator.py --help
    Linux: python3 data_generator.py --help
Пример запуска: python data_generator.py -o -n 100 200 300
